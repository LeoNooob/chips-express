# chips-express

基于express+mongodb+mysql搭建的后端服务



## 依赖

- [x] Express
- [x] MongoDB
- [x] MySQL
- [x] Faker数据

## usage

```shell
//clone本项目
git clone https://github.com/LeoNooob/chips-express.git
&
git clone git@github.com:LeoNooob/chips-express.git

//进入项目文件夹
cd chips-express
//安装依赖
npm i & npm install
//启动
npm start
```
